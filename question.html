Question-1: What are the different ways to select an element in the DOM?

document.getElementById("id") to select an element by its ID, document.getElementsByClassName("class") to get elements with a specific class, and document.getElementsByTagName("tag") to select elements by their tag name. Another way is document.querySelector("selector"), which selects the first matching element using a CSS selector, while document.querySelectorAll("selector") selects all matching elements. Additionally, document.getElementsByName("name") can be used to select elements by their name attribute.
